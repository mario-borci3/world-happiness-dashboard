# world-happiness-dashboard
All of the data, scripts, and files associated and needed to create the world happiness dashboard I made in the IBM Data Analyst Program.

# Project Goals
- Explore correlations between happiness score and key factors
- Visualize GDP per capita by country
- Compare happiness trends across regions

# Dashboard Features
- Correlation heatmap
- Scatter plot of GDP per capita vs Happiness Score
- Pie chart of Happiness Score by Region
- Choropleth map with GDP per capita and Healthy Life Expectancy tooltip

# Simply install python, download the csv and script, then run the script to generate all seperate visualizations and a complete dahsboard as html files.
